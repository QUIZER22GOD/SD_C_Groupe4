# SD_C_Groupe4

Bienvenue dans le dépôt officiel du **Groupe 4** – **Structure de Données (C)**.  
Ce dépôt rassemble l’ensemble des exercices pratiques, projets et travaux dirigés réalisés dans le cadre du cours.

## 📁 Organisation du dépôt

Chaque dépôt est fait dans une branche qui lui est propre afin de pouvoir cloner toute la branche lors des tests en local.
